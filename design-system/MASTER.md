# BizReply AI Design System

- Product direction: trustworthy, restrained B2B SaaS.
- Primary UI font: Inter. Sora is reserved for future marketing display use.
- Use semantic CSS tokens only; never hardcode colors in components.
- Prefer off-white backgrounds, white surfaces, deep green actions, and sparing gold accents.
- Controls are at least 44px high with visible labels, focus rings, disabled states, and inline errors.
- Use an 8px spacing rhythm, subtle borders, restrained shadows, and 12-16px radii.
- Motion is optional, meaningful, under 300ms, and respects reduced motion.
- All state views use shared loading, empty, and error components.
